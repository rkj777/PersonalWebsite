Dijkstra’s Algorithm Route Finder

A route finder like Google Maps that would use Dijkstra’s Algorithm to find the shortest route between two points on a map of Edmonton, Alberta, CA. There were two components that made up this project, the server and the client side. The server was written in python and would run Dijkstra’s algorithm on a directed graph to find the shortest path between two points. The client was implemented on an Arduino and used an LCD screen and joystick to allow the user to navigate a map of Edmonton and pick two points to route between.

Feistel Block Cipher

Created a server and client in Java that was able to communicate securely thorough the use of a Feistel block cipher. Using the Tiny Encryption Algorithm (TEA) messages were encrypted in blocks, which were sent to be decrypted based on shared keys. The project included both C and Java code so JNI was used to execute the C code.

Fault Tolerant Sorting

A system for fault-tolerant sorting using the recovery blocks technique. Two variants for sorting were written in two different languages, heapsort in Java and insertion sort in C. Using JNI to run C code in a Java program, a sorting program was created that would first run heapsort and then (in the case of a fault) run insertion sort.
